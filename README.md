# Library Management System

This project was developed during my first year of university while learning the fundamentals of **Object-Oriented Programming (OOP)** in **C++**.

The goal of the project was to practice core OOP concepts such as:

* classes and objects
* encapsulation
* constructors and methods
* operator overloading
* interaction between multiple classes

The implementation reflects my understanding of OOP concepts at that stage of my learning journey.

---

## Learning Objectives

Through this project I practiced:

* Designing multiple interacting classes
* Implementing a small management system using OOP principles
* Working with pointers and relationships between objects
* File input/output in C++
* Using STL containers such as `std::vector`

---

## Project Structure

* **Book** – represents a library book
* **Member** – represents a library user
* **Reservation** – manages book reservations
* **Fine** – represents penalties applied to members
* **Library** – the central class that manages books, members, reservations, and fines
* **main.cpp** – demonstrates and tests the functionality of the system
  
## Features

- Add and remove books from the library
- Register and manage library members
- Borrow and return books
- Create and process reservations
- Manage fines for members
- Display library statistics
